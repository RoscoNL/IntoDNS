# IntoDNS.ai

DNS & Security Tools Platform

## Concept

Een verzameling tools en services rondom DNS en security:

- **DNS Lookup** - A, AAAA, MX, TXT, NS, SOA, CNAME records
- **DNS Health Check** - Configuratie analyse en best practices
- **DNSSEC Validator** - Verificatie van DNSSEC configuratie
- **SPF/DKIM/DMARC Checker** - Email security records validatie
- **DNS Propagation** - Wereldwijde DNS propagatie check
- **Reverse DNS** - PTR record lookups
- **Whois Lookup** - Domein registratie informatie
- **SSL/TLS Checker** - Certificaat analyse
